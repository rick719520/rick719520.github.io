Vue 3 待办事项管理系统

基于 Vue 3 + Vite 构建的待办事项管理系统，使用 Element Plus 组件库。

项目技术栈

  • Vue 3 + Vite
  • Vue Router 4
  • Pinia
  • Axios
  • Element Plus

功能特点

  • 用户登录/登出
  • 任务总览和统计
  • 任务列表展示与搜索
  • 任务详情查看与编辑
  • 新增任务
  • 分类管理
