JavaScript入门：DOM操作与事件处理

DOM基础概念

DOM（Document Object Model，文档对象模型）是HTML和XML文档的编程接口。它将网页表示为一个树状结构，其中每个节点都是文档的一部分，如元素、属性或文本。
